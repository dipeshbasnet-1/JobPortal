# JobPortal - MernStack

A full-featured MERN stack Job Portal web application where job seekers can apply for jobs and employers can post and manage job listings.

## Features
 
👤 Authentication
  • User registration & login (JWT based)
  • Role-based access (Admin, Job Provider, Job Seeker)
  • Secure protected routes

💼 Job Management
  • Create job postings (Providers)
  • View all available jobs
  • View single job details
  • Apply for jobs (Seekers)
  • View posted jobs (Providers)
  • Delete / Manage jobs

📊 Dashboards
  • Job Seeker Dashboard
  • Job Provider Dashboard
  • Admin Controls

💳 Payment Integration
  • Payment route setup
  • Payment model & controller
  • Backend payment handling logic

📧 Email Service
  • Email helper service
  • Send email functionality integrated

## Tech Stack

# Frontend
React (Vite) | Axios | Tailwind CSS | React Router | Redux Toolkit

# Backend
Node.js | Express.js | PostgreSQL | Sequelize ORM | JWT Authentication | REST API Architecture
